4) Why isn’t it possible to see the first five rows of a table?
Answer-> 
Rows of table are scattered (fragmented) all over the DB Server HD. when you insert into a table wherever the system finds the free space in DB server HD.it will store the row there. thus it is not possible to view the first or last five rows of a table.
